学习使用Hexo+github搭建并部署个人博客

基础准备

在使用Hexo前需要在电脑上分别安装Node.js和git，安装步骤可直接上度娘搜索，然后再github上注册一个账号

Node.js

网址：http://nodejs.cn/

直接点击长期支持版本即可，它默认会给我们安装好Node.js和npm（Node.js的包管理工具），下载安装包后，除了修改一下安装目录就好了（默认C盘，但是C盘不建议放太多不是系统文件的文件）无脑下一步就好了。

git的安装和Node.js也是一样的。

Windows系统可能需要自己设置环境变量，将对应安装目录添加到系统环境变量即可，浏览器搜索下对应方法即可。

安装完后，win+R打开一个搜索框，输入cmd然后打开终端，输入node -v npm -v 和 git --version查看一下自己的安装版本

安装Hexo

建议现在除C盘以外的任意一个盘里先新建一个文件夹用来存放一些关于Hexo的文件（例如未来搭建的网页文件等）

按上面的方法进入终端后，输入D:进入到D盘，在输入cd 你创建的文件名就可以进入到你创建的文件夹下进行命令操作，

输入下面代码进行安装

npm install -g hexo-cli，结束后，在输入hexo -v查看一下版本信息，确保安装正确。

初始化博客

hexo init Myblog 生成一个关于博客的文件夹名字叫Myblog

安装依赖

npm install

预览

hexo s输入命令后会有一个这个链接http://localhost:4000/，最初是的个人博客搭建好了，但只可以在自己的电脑上进行访问

上传到github

先到github注册一个用户，接着创建一个仓库，复制仓库链接，用编辑器打开Myblog文件夹，找到文件名为_config.yml打开，找到末尾，添加一下代码

repo中为刚刚复制的仓库HTTPS链接，保存。

打开终端，进入到文件夹Myblog，输入命令hexo clean清除缓存，接着hexo g 生成一下我们的文件，hexo d上传文件到github

这样就基本结束了，想要美化博客主题的看下一篇文章。
